========================================
       WeJZ Client - Setup Guide
========================================

REQUIREMENTS:
  Python 3.10 or newer
  Download: https://www.python.org/downloads/
  
  [!] Check "Add Python to PATH" during install!

========================================
           SETUP
========================================

1. Install Python (check "Add to PATH")

2. Double-click "install.bat"

3. Double-click "start.bat" to play!

========================================
        GETTING STARTED
========================================

1. Click [?] LOGIN (bottom left)
2. Create account (right side)
3. Go to FRIENDS tab
4. Add friends by username!

========================================
           UPDATES
========================================

Updates are automatic! When a new 
version is available, you'll see a 
notification at the top. Just click
"UPDATE NOW" and restart.
